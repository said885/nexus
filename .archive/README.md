# Development Archive

This directory contains working notes and session summaries from the development process.

These files are preserved for historical reference but are not required for running or deploying NEXUS.

## Contents

- `SESSION_*` — Session summaries and completion reports
- `FILES_CREATED_*.txt|.md` — Lists of files created in each development phase
- `IMPLEMENTATION_*.md` — Implementation checklists and progress reports
- `README_SESSION_*.txt` — Session-specific README files
- `QUICK_START_*.md` — Early quick-start guides
- `.` — Various working documents

## For Users

If you're setting up NEXUS, start with the main [README.md](../README.md) instead.

## For Developers

If you want to understand the evolution of NEXUS, start with:

1. `SESSION_7_COMPLETION.md` — Latest session summary
2. `IMPLEMENTATION_COMPLETE.md` — Final implementation status
3. Ring older sessions backwards in time

## Archive Policy

We keep this archive for:
- Historical reference
- Understanding design decisions
- Tracing the evolution of features

This directory is not part of the git repository (see `.gitignore`).
